SkyScanner WP Widget 1.0.0

Installation

1. Install SkyScanner WordPress Plugin sky-scanner-widget.zip to your website. Don't know how? Please learn here.

2. Go to SkyScanner plugin page in WordPress admin menu.

3. Adjust all settings.

Use Sky Scaner as a WordPress Widget

To use Sky Scanner as a WordPress widget, go to Appearance -> Widgets, and drag Sky Scanner WordPress widget to any widget section. Next, select one of available SkyScanner widgets from the list.

Use widget as a WordPress Shortcode

Example of the Sky Scanner shortcode:

[skyscannerwidget]
